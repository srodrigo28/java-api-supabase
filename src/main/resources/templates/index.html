Java API com Spring Boot

Versão v1

POST /cadastrar Cria um novo registro

URL Completa: localhost:8080/api/serie/cadastrar

Body (JSON):
{
    "nome": "Java como programar 22",
    "descricao": "Ano 2024"
}
GET /listar Retorna todos os registros

URL Completa: localhost:8080/api/serie/listar

Response (JSON):
[
    {
        "id": 5,
        "nome": "Java como programar 17",
        "descricao": "Ano 2019"
    },
    {
        "id": 6,
        "nome": "Java como programar 20",
        "descricao": "Ano 2022"
    }
]
PUT /alterar Atualiza um registro existente

URL Completa: localhost:8080/api/serie/alterar

Body (JSON):
{
    "id": "3",
    "nome": "Java 6 antigamente",
    "descricao": "Criando jdbc"
}
DELETE /remover/{id} Remove um registro pelo ID

URL Completa: localhost:8080/api/serie/remover/5

Response:
[] 
// Status 200 OK ou 204 No Content
